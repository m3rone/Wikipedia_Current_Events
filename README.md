# Wikipedia Current Events RSS Feed

An RSS feed of Wikipedia [current events](https://en.wikipedia.org/wiki/Portal:Current_events) tab, delayed by one day.

[RSS feed](https://github.com/m3rone/Wikipedia_Current_Events/raw/refs/heads/main/rss/index.xml)

The feed items do not include the text of the current events, but you can add `div.current-events-content.description` to the retrieval settings of your RSS client of choice

Forked from [CodebergIsBetter/Wikipedia_Current_Events](https://github.com/CodebergIsBetter/Wikipedia_Current_Events), which has a delay of 2 days.
